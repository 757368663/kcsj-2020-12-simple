材料表 & 材料分类表

  • 装修材料分类

    • 墙体材料

      • 腻子，壁纸，乳胶漆，墙面砖

    • 顶部材料

      • 石膏板，PVC板，透光板，顶板

    • 装饰线

      • 踢脚线，护墙板，檐口线脚

    • 地面材料

      • 地砖，木质地板，地毯，找平材料

    • 链接材料

      • 紧固件，粘接剂，连接件

  • 库表关系

    • 装修材料分类与具体材料为一对多关系，因此材料表属性一条外键为材料分类id

  • 库表字段

    • 材料分类库

      • id ， name（分类名称），person（录入人），create_date（创建时间）

    • 材料库

      • id ，name（材料名称），count（数量），person（录入人），create_date(创建时间)，type（材料类型，外键），remark（备注），phone_number (联系人电话)

       

 

出入库订单表

  • 库表关系

    • 订单与材料绑定，为多对一关系，因此订单表属性一条外键为材料id

  • 库表字段

    • id，name（材料名称），meterial_id(材料id)，count（出入库数量），person（操作人），reason（操作原因），inb（是否为入库），create_date （创建时间）
